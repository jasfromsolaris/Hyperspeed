# Hyperspeed

Internal team workspace: board, chat, files, IDE-style editing, and automations—shipped as a **Docker Compose** stack we run on infrastructure we control.

- **Quick start:** from the **repository root**, run **`docker compose up --build`** (default UI: **http://localhost:18080**). Configuration lives in the root **[`.env`](.env)** (`KEY=value` lines, no `#` comments—works well with env import UIs). Operational detail: **[README_SELF_HOST.md](README_SELF_HOST.md)**.
- **Vision and product boundaries:** **[VISION.md](VISION.md)**.
- **Domains and TLS:** **[docs/ops/custom-domains-and-subdomains.md](docs/ops/custom-domains-and-subdomains.md)**.
